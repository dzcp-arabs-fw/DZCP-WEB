Introduction

DZCP is a powerful event-driven framework designed to simplify game development with a modular structure and built-in support for custom events.

Available Events

  • OnPlayerEscape: Triggered when a player escapes the facility.
  • OnPlayerBan: Triggered when a player is banned.
  • OnScpEscape: Triggered when an SCP escapes.
  • OnGateOpened: Triggered when a gate is opened.
