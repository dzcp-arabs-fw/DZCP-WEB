What is DZCP?

DZCP is a modern, event-Games framework for game development. It simplifies the process of creating custom events and provides a modular structure for scalability and flexibility.

Key Features

  • Event-Driven Architecture.
  • Easy Integration with Plugins.
  • Highly Customizable.
  • Support for Player and SCP Events.
  • Easy and fast.
  • more power.
  • more effects.
  • Customroles.
  • Customitems.
